Команди меню "Файл"

  • Створити - створює новий пустий файл. Якщо у полі вже введено який-небуть текст, програма запропонує його зберегти.
  • Відкрити - відкриває файл. Якщо у полі вже введено який-небуть текст, програма запропонує його зберегти.
  • Зберегти - зберігає текст в поточний відкритий файл.
  • Зберегти як... - зберігає текст у новий файл. 
  • Статистика - відображає статистику введеного тексту:
    • кількість рядків
    • кількість слів
    • кількість символів
  • Вихід - виходить із програми. Якщо у полі вже введено який-небуть текст, програма запропонує його зберегти.
